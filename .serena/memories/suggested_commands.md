# 推奨コマンド

## セットアップ
```bash
# 依存パッケージのインストール
pip install -r requirements.txt
```

## 実行
```bash
# Streamlitアプリの起動
streamlit run streamlit_app.py
```

## デプロイ
- Streamlitのマイページからデプロイ: https://streamlit.io/

## 開発時の確認
```bash
# Pythonの構文チェック
python -m py_compile streamlit_app.py

# パッケージの確認
pip list
```

## Git操作
```bash
# 変更の確認
git status

# コミット
git add .
git commit -m "メッセージ"

# プッシュ
git push origin main
```
